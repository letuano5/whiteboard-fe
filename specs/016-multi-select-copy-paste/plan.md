# Implementation Plan: Multi-select + Duplicate/Copy-Paste (P2-08)

**Branch**: `feat/online-room` | **Date**: 2026-06-27 | **Spec**: [spec.md](spec.md)

**Input**: Feature specification from `specs/016-multi-select-copy-paste/spec.md`

## Summary

Extend the existing single-element selection to support multi-element selection (rubber-band marquee + Shift-click), bulk operations (move, style, delete), and clipboard operations (Ctrl/Cmd+D duplicate, Ctrl/Cmd+C copy, Ctrl/Cmd+V paste). The store already has `selectedIds: string[]`; the marquee state exists but is unconnected. All mutations use the existing `updateElements` batch pipeline (one history entry per bulk operation).

## Technical Context

**Language/Version**: TypeScript 6.x + React 19.x

**Primary Dependencies**: Zustand 5.x (interaction store), SVG (marquee rendering), mutation-pipeline.ts (batch ops)

**Storage**: Clipboard = in-memory array on `interaction.store`; no persistence

**Testing**: Vitest 4.x

**Target Platform**: Browser (desktop; touch out of scope)

**Project Type**: Web application ‚Äî SVG whiteboard

**Performance Goals**: Bulk move ‚â§ 50 shapes with no perceptible lag

**Constraints**: Each multi-shape operation must be a single undo step; no OS clipboard

**Scale/Scope**: User-facing selection + clipboard; affects select-tool, SvgLayer, DetailPanel, Whiteboard

## Design Details

### 1. Interaction Store Extensions

New fields in `InteractionState`:

```ts
clipboard: Element[] | null;     // deep-cloned snapshot at Ctrl+C time
pasteOffset: number;             // incremented per Ctrl+V; offset = pasteOffset * 10
draftElements: Element[];        // transient positions during multi-drag
```

New actions: `setClipboard`, `setPasteOffset`, `setDraftElements`.

### 2. Mutation Pipeline ‚Äî `createElements` batch

```ts
export function createElements(drafts: ElementDraft[]): Element[]
```

Creates all elements in one `MutationEvent` (`type: 'create'`, `before: []`) ‚Üí history capture sees ONE entry ‚Üí single undo step.

### 3. Select Tool ‚Äî Pointer Logic Changes

**`onSelectPointerDown(worldPt, shiftKey)`**

1. Hit-test topmost non-deleted element.
2. Hit + shiftKey ‚Üí toggle id in selectedIds.
3. Hit + no shiftKey + id in selectedIds ‚Üí keep selection; start drag.
4. Hit + no shiftKey + id not in selectedIds ‚Üí replace selection with [id]; start drag.
5. Miss ‚Üí clear selection (unless shiftKey); start marquee drag.

**`onSelectPointerMove(worldPt)`**

- Marquee active ‚Üí update `marquee = normalizedRect(dragStart, worldPt)`.
- Dragging + selectedIds.length > 1 ‚Üí update `draftElements` with (dx, dy) applied to all.
- Dragging single ‚Üí existing draftElement logic.

**`onSelectPointerUp(worldPt)`**

- Marquee ‚Üí compute intersection; `setSelectedIds(matchingIds)`; clear marquee.
- Multi-drag committed ‚Üí `updateElements(draftElements patches)` (one undo step); clear draftElements.
- Single drag ‚Üí existing patchElement.

### 4. SvgLayer Multi-selection Visual

`selectedIds.length > 1`:
- Render union bounding box `<rect>` (blue dashed stroke, no handles).

`selectedIds.length === 1`:
- Existing `SelectionOverlay` with handles.

Marquee in-progress (`marquee !== null`):
- Render `<rect>` from `marquee` (light blue fill at 10% opacity, dashed border).

### 5. DetailPanel Multi-selection

`selectedIds.length > 1`:
- Show: Stroke color, Fill color, Stroke width, Stroke style, Opacity.
- On change: `updateElements(selectedIds.map(id => ({ id, patch: { props: { ... } } })))`.
- Hide: Transform (angle), Text section.

### 6. Keyboard Shortcuts

| Key | Action function |
|-----|----------------|
| Delete / Backspace | `deleteElements` ‚Üê already works |
| Ctrl/Cmd+D | `onDuplicateSelected()` |
| Ctrl/Cmd+C | `onCopySelected()` |
| Ctrl/Cmd+V | `onPasteSelected()` |

### 7. Duplicate / Copy / Paste

**`onDuplicateSelected()`**: clones selected elements (+10, +10 offset) ‚Üí `createElements` ‚Üí select copies.

**`onCopySelected()`**: deep-clone selected ‚Üí `setClipboard`; `setPasteOffset(0)`.

**`onPasteSelected()`**: offset = `(pasteOffset + 1) * 10`; build drafts from clipboard; `createElements` ‚Üí select copies; `setPasteOffset(pasteOffset + 1)`.
