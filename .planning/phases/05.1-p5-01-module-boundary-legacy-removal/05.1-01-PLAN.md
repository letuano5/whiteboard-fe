## Implementation Slices

1. Add backend sync module contracts and `executeSyncCommand`.
   - Support `legacy-element-update` compatibility command for current realtime edits.
   - Support `native-file-import` compatibility command for saved-room native import.
   - Keep repository/autosave/room-memory mutation inside the module.
   - Verification: `pnpm --filter whiteboard-be test -- execute-sync-command`.

2. Rewire realtime `ELEMENT_UPDATE` handler.
   - Keep auth/permission/session responsibilities in the handler.
   - Delegate accepted saved-room mutation to `executeSyncCommand`.
   - Broadcast the committed compatibility payload returned by the sync module.
   - Verification: `pnpm --filter whiteboard-be test -- element-update`.

3. Rewire native-file import.
   - Keep HTTP payload validation and permission check in the router/service.
   - Delegate document replacement persistence to `executeSyncCommand`.
   - Verification: `pnpm --filter whiteboard-be test -- native-file-import`.

4. Add AC coverage and legacy scope comments.
   - Tag each acceptance test with `@covers AC-n`.
   - Add comments that legacy whole-element compatibility is not the final P5 saved-room authority.
   - Run AC coverage script and relevant backend test/typecheck commands.
