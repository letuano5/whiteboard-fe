migration support for persisted idempotency, shared transient/resendable delivery hints, and room
repository recovery load state with safe clock conversion.

## Implementation Slices

1. Add `Room.roomEpoch`, `Room.processedRequestHistoryStartsAtClock`, and `ProcessedRequest`
   schema/migration fields.
   - Verification: `pnpm --filter whiteboard-be prisma validate` and
     `pnpm --filter whiteboard-be prisma generate`.

2. Add optional shared command persistence hints with safe defaults:
   durable/resendable/store `ProcessedRequest` unless an intermediate patch explicitly says
   transient/non-resendable/relaxed.
   - Verification: `pnpm --filter whiteboard-be test -- shared-contract-invariants`.

3. Expand `loadRoomElements` to return `roomEpoch`, tombstone IDs, slot clocks, and safe clock
   conversions for recovery/reconnect code.
   - Verification: `pnpm --filter whiteboard-be test -- room-repository`.
