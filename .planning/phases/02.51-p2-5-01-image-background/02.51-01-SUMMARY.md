# Summary 02.51-01: Add Image / Background Map Support

## Accomplishments

- Added `imageShapeUtil` and registered `image` elements in the SVG shape registry.
- Added a compact toolbar image control with URL insertion and local file upload converted to data
  URL.
- Inserted images are committed through the mutation pipeline, selected immediately, and patched
  behind existing visible elements to support background map use.
- Added acceptance coverage for SVG rendering, URL/data URL insertion, background z-order, and
  select-tool move/resize behavior.
- Tightened image resize behavior so selected images expose only corner handles, edge handles are
  ignored, and corner resizing preserves the existing image aspect ratio so the bbox stays fitted to
  the image.
