<objective>
Deliver the Merge/Unmerge command layer for the "merge items" MVP: a shared-`groupId` unit of
operation created and cleared from the context menu and keyboard, with group-aware delete cascade
and a fixed copy/duplicate `groupId` remap. This is slice (a) of the D-04-mandated 3-slice
delivery — it builds the pure group-resolution foundation (`group.ts`) that slices (b) and (c)
extend, and wires the two new commands into the existing context menu and shortcut dispatch.

Purpose: Establish `groupId` grouping as a persistent, undoable operation reusing the existing
mutation pipeline and `grouping.groupId` sync slot — no new shared type, no new WS event (AC-13).
Output: `group.ts`, `merge.ts`, modified `clipboard.ts` / `ContextMenu.tsx` / shortcut modules,
plus Wave 0 test files for the whole phase's `select/__tests__` suite.
</objective>
